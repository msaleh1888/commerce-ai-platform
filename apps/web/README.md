# Web

Next.js frontend for the commerce AI platform.

## Local Development

From `apps/web`:

```bash
npm install
npm run dev
```

The app runs at `http://localhost:3000` by default.

## Checks

```bash
npm run typecheck
npm run build
```

## Scope

This M1 skeleton configures Next.js, TypeScript, and Tailwind CSS. It leaves the app shell, navigation, design system primitives, lucide icons, and shadcn-style components for later frontend issues.
